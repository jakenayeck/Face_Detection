# Face_Detection
This is a project to explore how to detect faces using Haar Cascades. 
To run: 
 `python face_detection.py`
To quit:
 Press q.


##Technolgies: 
* Python
* OpenCV
* NumPy

##Installation Guide:
* OpenCV (for mac)
  * `brew tap homebrew/science`
  * `brew install opencv`
* Numpy
 * `pip install numpy`
